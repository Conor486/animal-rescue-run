# 🐾 Animal Rescue Run!

A side-scrolling platformer game built for kids — powered by **Phaser 3** game engine with procedural pixel art sprites!

## 🎮 Features

- **Phaser 3.90** — Professional HTML5 game engine with WebGL rendering & arcade physics
- **5 Unique Worlds** — Sunny Meadow, Mystic Forest, Crystal Caves, Snowy Peaks, Volcano Island
- **Procedural pixel art** — All sprites, tiles, backgrounds & UI generated at runtime
- **Real physics** — Gravity, variable jump height, momentum, enemy stomping
- **3 Enemy types** — Slimes, bats, and spike balls with unique behaviours
- **8 Rescuable animals** — Bunny, fox, bird, frog, cat, panda, penguin, parrot
- **Touch-friendly** — Works on tablets and phones with on-screen controls
- **Keyboard support** — Arrow keys / WASD + Space
- **Scene system** — Menu, gameplay, level complete, game over, victory screens
- **Parallax scrolling** — Multi-layer backgrounds for depth
- **Particle effects** — Sparkles on rescue, confetti on victory

## 🚀 Deploy

This is a static single-file game. Deploy anywhere:

```bash
# Vercel
npx vercel --prod

# Or just open index.html in a browser
```

## 📱 Controls

| Action | Keyboard | Touch |
|--------|----------|-------|
| Move Left | ← or A | ◀ button |
| Move Right | → or D | ▶ button |
| Jump | ↑ / W / Space | JUMP button |
| Variable Jump | Hold key longer | Hold button longer |
